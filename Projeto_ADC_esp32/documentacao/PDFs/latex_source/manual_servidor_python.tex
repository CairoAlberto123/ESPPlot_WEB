\section{Configuração de Filtros}
Fórmula do filtro passa-baixa:
\[
H(s) = \frac{1}{\sqrt{1 + \left(\frac{f}{f_c}\right)^{2n}}}
\]
Onde:
\begin{itemize}
\item \(f_c\): Frequência de corte
\item \(n\): Ordem do filtro (padrão=5)
\end{itemize}
